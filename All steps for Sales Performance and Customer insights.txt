2. Import Dataset into Power BI:
Steps:
Open Power BI Desktop
Click Get Data → Text/CSV
Load:
customers.csv
products.csv
sales.csv
Click Transform Data (Power Query)

3. Data Cleaning (Power Query)
Do this:
Check for null values
Ensure correct data types:
Date → order_date, signup_date
Number → price, quantity, revenue
Rename columns if needed

👉 Click Close & Apply

4. Data Modeling (Very Important)

Go to Model View and create relationships:

sales.customer_id → customers.customer_id
sales.product_id → products.product_id

👉 Relationship Type: Many-to-One

5. Create Key KPIs (DAX Measures)
1. Total Revenue
Total Revenue = SUM(sales[revenue])

2. Total Orders
Total Orders = COUNT(sales[order_id])
3. Total Customers
Total Customers = DISTINCTCOUNT(customers[customer_id])
4. Monthly Revenue
Monthly Revenue = 
CALCULATE(
    SUM(sales[revenue]),
    MONTH(sales[order_date])
)
5. Customer Retention (Basic)
Repeat Customers = 
CALCULATE(
    DISTINCTCOUNT(sales[customer_id]),
    FILTER(
        VALUES(sales[customer_id]),
        CALCULATE(COUNT(sales[order_id])) > 1
    )
)

7. Customer Segmentation (Important for Interview)
Create Column:
Customer Type = 
IF(
    CALCULATE(SUM(sales[revenue])) > 10000,
    "High Value",
    IF(
        CALCULATE(SUM(sales[revenue])) > 5000,
        "Medium Value",
        "Low Value"
    )
)
👉 Use this in charts:

Customer distribution
